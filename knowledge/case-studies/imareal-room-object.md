# Use Case: REALonline Rauminventar-Analyse Dashboard

## Kontext

Mittelalterliche Rauminventare aus der REALonline-Datenbank für Sachkulturforscher explorierbar machen. Daten aus österreichischen und norditalienischen Adelshaushalten. Raum-Objekt-Beziehungen aus dem Spätmittelalter.

- Stand: 1. Promptotyping Iteration
- Zeitinvestition: 5 h
- Repository: github.com/chpollin/imareal-room-object
- Demo: chpollin.github.io/imareal-room-object

## LLMs und Tools

- Claude Opus 4.1 (Claude Desktop) für konzeptuelle Arbeit
- Claude Sonnet 4.5 (Claude Code) für Implementation

## Notizen zum Prozess

- Treemap-Visualisierung trotz mehrfacher Feedbackschleifen problematisch
- JOURNAL.md erstmals als aktives Dokument (Claude dokumentierte selbstständig)
- Promptotyping Systems-Prompt verwendet
- Journal dokumentiert Warum, Commits fixieren Was
- GitHub Commits als Savepoints zwischen Phasen
- Visuelle Rückkopplung durch Screenshots
- Ohne expliziten MVP direkt implementiert
